# Greedy
###### tags = `Algorithm`

Greedy的概念很簡單，每一步都採用當前看起來最好的選擇，真的能使最終答案最好嗎?
如果可以，那就可以使用Greedy演算法

### Change Coin

* 去銀行換錢時，通常會拿到數量最少的紙鈔或硬幣，假設現在要換到n元，如何讓硬幣跟紙鈔的數量最少?
* 每一步都採用當前看起來最好的選擇，真的能使最終答案最好嗎?
    * 都拿<n裡，幣值最大的
* 如果台幣的硬幣及紙鈔面額改變，還能使用greedy嗎?
